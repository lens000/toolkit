Fit Ellipse
	There are two ways to fit ellipse. The first way is based on algebraic fitting and the second
	way is based on geometric fitting(best fitting).

	If you want to use algebraic fitting, you can download the paper from this websit http://autotrace.sourceforge.net/WSCG98.pdf.
	
	
	If you want to use geometric fitting, you can download the paper from this websit 
	http://autotrace.sourceforge.net/WSCG98.pdf.https://www.sciencedirect.com/science/article/abs/pii/S0031320300001527

Note:
    1.transform point system
    2.newton iteration (multivariable)
    3.gauss-newton iteration (nonlinear least squares)
    4.svd
